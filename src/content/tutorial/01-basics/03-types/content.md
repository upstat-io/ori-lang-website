# Basic Types

Ori has four primitive types:

| Type    | Description          | Example           |
|---------|----------------------|-------------------|
| `int`   | Integer numbers      | `42`, `-7`, `0`   |
| `float` | Floating point       | `3.14`, `-0.5`    |
| `bool`  | Boolean              | `true`, `false`   |
| `str`   | Text strings         | `"hello"`         |

You can convert between types using built-in functions:

- `str(42)` converts an integer to `"42"`
- `str(3.14)` converts a float to `"3.14"`
- `str(true)` converts a bool to `"true"`

## Your task

Create one variable of each type and print them all on separate lines. The output should be:

```
42
3.14
true
hello
```
