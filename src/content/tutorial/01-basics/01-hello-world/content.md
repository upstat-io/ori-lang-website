# Hello World

Every Ori program starts with a `@main` function — this is the entry point, just like `main` in C or Rust.

The `print` function writes text to the output. In Ori, function arguments are **named** — you write `msg:` before the value:

```ori
print(msg: "Hello, World!")
```

## Your task

Change the message so that it prints **"Hello, Ori!"** instead of "Hello, World!".

Then click **Run** (or press `Ctrl+Enter`) to check your answer.
